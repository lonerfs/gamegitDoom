В основе лежат готовые файлы, нам надо писать свой редактор и уровни. Для понимания структуры используем это.
Юзаем готовое, пока пишем движок, потом создаем редактор и свой контент!

Текущее состояние проекта (на сегодня)

У нас есть:

    Настроенный проект на C++ с SDL3 (окно, рендерер).

    Загрузчик WAD (WadLoader) – читает файлы .wad, получает список лумпов.

    Структуры карты Doom: Vertex, Linedef, Sidedef, Sector, Thing.

    Класс DoomMap, который загружает карту (E1M1) из freedoom1.wad и раскладывает по векторам.

    2D-отрисовка карты в окне (вид сверху, линии стен).

    В консоль выводится статистика загрузки.

Задача минимум:
Движок: 3D-рендеринг и управление (2 недели)
Цель: превратить 2D-вид в игру от первого лица с возможностью ходить по карте и видеть стены.
Задачи:
1. Камера и управление – добавить класс `Player` с координатами (x, y), углом обзора. Реализовать поворот мыши и движение WASD.
2. Рейкастинг – для каждого столбца экрана пускать луч, определять ближайшую стену и расстояние до неё.
3. Отрисовка стен – рисовать вертикальные полосы высотой, обратно пропорциональной расстоянию. Пока без текстур, просто цветом (разные цвета для разных сторон).
4. Коллизии – запретить проходить сквозь стены (проверка пересечения с линиями).
5. Поддержка нескольких уровней – загружать разные карты по команде (например, E1M2).

+ к минимуму:

Текстуры, пол, потолок и спрайты (2–3 недели)
Цель: добавить текстуры на стены, пол, потолок и отображать предметы (спрайты).
Задачи:
1. Текстуры стен – загружать из WAD патчи текстур. Для каждого столбца стены брать соответствующую текстурную координату и рисовать пиксели.
2. Пол и потолок – реализовать рендеринг плоскостей (горизонтальных поверхностей) с текстурами. Использовать метод, аналогичный Doom (порталы или построчный).
3. Спрайты – загрузить спрайты предметов (аптечки, патроны) из WAD. Отображать их как биллборды с учётом расстояния и сортировкой по глубине.
4. Анимация – для спрайтов, у которых несколько кадров (мигающие предметы), добавить смену кадров по времени.
